var stinuum = new Stinuum(viewer);

## Methods

* show(id, propertyName)
If Moving Feature whose id is in the features, then show TemporalMap of Moving Feature.
&nbsp;

Example:
```js
var id = "mf_001";
var name = "speed";

stinuum.temporalMap.show(id, name);
```
